# CS 330 – Computer Graphics

This repository contains my final OpenGL 3D graphics project.

## Features
- Low-polygon 3D scene built using OpenGL
- Composite object (coffee mug made from multiple primitives)
- Texturing with 1024x1024 royalty-free textures
- Two point light sources (Phong shading model)
- WASD + QE camera movement
- Mouse yaw and pitch control
- Scroll wheel movement speed adjustment
- Perspective and orthographic projection toggle (P / O)

## Controls
WASD – Move forward/back/left/right  
Q / E – Move down/up  
Mouse – Look around  
Scroll – Adjust speed  
P – Perspective view  
O – Orthographic view

## Reflection
How do I approach designing software?

I approach software design by first identifying the intended outcome and breaking the system into manageable components. For this project, I visualized the full tabletop scene before building individual objects. I focused on structure, transformations, and visual balance to ensure the final result was cohesive and realistic.

What new design skills did I develop?

This project strengthened my spatial reasoning, lighting configuration, and texture mapping skills. I learned how object placement, scale, UV mapping, and light positioning significantly impact realism and clarity in a 3D environment.

What design process did I follow?

I followed an iterative process. I began with primitive object placement, then layered in textures, lighting, and camera controls. Each milestone improved the visual and structural quality of the scene.

How do I approach developing programs?

I develop programs incrementally. I implement one feature at a time, test it thoroughly, and refine before adding complexity. This reduces debugging time and improves code stability.

How did iteration influence development?

Iteration was essential. Each refinement improved lighting balance, object alignment, and camera interaction. Testing projection modes and adjusting shader behavior enhanced the final presentation.

How has my development approach evolved?

Throughout the milestones, I became more comfortable structuring OpenGL programs modularly. Separating SceneManager and ViewManager improved organization and maintainability.

How does computer science support my goals?

Computer science builds strong problem solving skills and technical knowledge that prepare me for software development roles. It allows me to design interactive systems and visual environments effectively.

How do computational graphics help academically and professionally?

Computational graphics strengthen understanding of linear algebra, transformations, and rendering systems. Professionally, these skills apply to game development, simulation, visualization, UI design, and interactive software systems.
